# Как поднять локальный кластер Kafka

1. Из репозитория 
```console
make kafka-up
```

2. Зайдите на адрес ``localhost:8090``,
там вас ждет UI в котором будет уддобно смотреть,
что отправляет приложениие в Kafka

# Как выключить локальный кластер Kafka
1. Из репозитория 
```console
make kafka-down
```
